# Customer Analytics: RFM, Churn Prediction & CLV

## Project Overview

This project analyzes customer purchasing behavior using retail transaction data. The goal is to identify valuable customer segments, predict churn and estimate customer lifetime value (CLV).

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## Analysis Performed

### Data Cleaning
- Handled missing values
- Converted data types
- Created Revenue metric

### Exploratory Data Analysis
- Customer analysis
- Product performance analysis
- Country-wise revenue analysis
- Monthly revenue trends

### RFM Customer Segmentation

Analyzed customers based on:
- Recency (recent purchases)
- Frequency (number of purchases)
- Monetary value (spending)

Customer segments:
- Champions
- Loyal Customers
- Potential Loyalists
- At Risk
- Lost Customers

### Churn Prediction

Built a Logistic Regression model to identify customers likely to churn based on customer behavior.

### Customer Lifetime Value (CLV)

Estimated customer value based on purchasing behavior and categorized customers into:
- Low CLV
- Medium CLV
- High CLV

## Key Outcomes

- Identified high-value customers
- Segmented customers based on behavior
- Predicted potential churn customers
- Generated insights for customer retention strategies
